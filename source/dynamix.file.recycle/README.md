<span class="dfr-plugin-description-en"><strong>Dynamix File Recycle Bin</strong><br>Adds a guarded, per-disk and per-ZFS-dataset Recycle action to Unraid's official built-in Dynamix File Manager. It does not intercept deletion from SMB, terminal commands, containers, applications, or third-party file managers. Do not manually move, rename, delete, or edit anything inside <code>.RecycleBin</code>.</span>
<span class="dfr-plugin-description-zh"><strong>Dynamix 文件回收站</strong><br>为 Unraid 官方内置 Dynamix 文件浏览器增加按磁盘、按 ZFS dataset 隔离的安全回收操作。它不会拦截 SMB、终端命令、容器、应用程序或第三方文件管理器执行的删除。请勿手动移动、重命名、删除或编辑 <code>.RecycleBin</code> 内的任何内容。</span>
